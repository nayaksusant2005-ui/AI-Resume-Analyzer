# 🚀 AI Resume Analyzer

AI Resume Analyzer is a Flask-based web application that analyzes resumes and provides an ATS (Applicant Tracking System) score along with skill detection, missing skills analysis, improvement suggestions, and downloadable PDF reports.

## 📌 Features

* Resume Upload
* ATS Score Calculation
* Skill Detection
* Missing Skill Analysis
* Resume Improvement Suggestions
* Dynamic PDF Report Generation
* User-Friendly Interface

## 🛠️ Technologies Used

* Python
* Flask
* HTML
* CSS
* ReportLab
* CSV

## ⚙️ Installation

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

4. Open browser

http://127.0.0.1:5000
